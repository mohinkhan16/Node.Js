# 🚀 Task Manager API

A simple REST API built with Express.js for managing daily tasks.

---

## ✨ Features

✅ Get All Tasks  
✅ Get Single Task  
✅ Add New Task  
✅ Update Existing Task  
✅ Delete Task  
✅ Custom Error Handling  
✅ JSON Middleware Support  

---

# 🛠 Tech Stack

- Node.js
- Express.js
- JavaScript

# Car CRUD API 🚗

A simple Car Management CRUD API built using Node.js and Express.js with custom error handling middleware.

---

## 📌 Features

- Get all cars
- Get car by ID
- Add new car
- Update car details
- Delete car
- Custom Error Handling Middleware
- REST API Implementation
  
---
## 🛠️ Technologies Used

- Node.js
- Express.js
- JavaScript
